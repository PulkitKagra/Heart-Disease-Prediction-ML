# ❤️ Heart Disease Prediction - Machine Learning Project

An end-to-end Machine Learning web application that predicts the likelihood of heart disease based on patient medical attributes.

This project includes data preprocessing, model training, evaluation, and deployment using Streamlit.

## 📌 Problem Statement

Heart disease is one of the leading causes of death worldwide.  
The goal of this project is to build a Machine Learning classification model that can predict whether a person is at risk of heart disease based on medical parameters.

---

## 📊 Dataset Information

- Source: UCI Heart Disease Dataset
- Type: Binary Classification
- Target Variable:
  - 0 → No Heart Disease
  - 1 → Heart Disease

### Features Used:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise-Induced Angina
- Oldpeak (ST depression)
- ST Slope

---

## 🧠 Model Details

- Algorithm Used: **K-Nearest Neighbors (KNN)**
- Data Split: 80% Training / 20% Testing
- Accuracy Achieved: **87.5%**

---

## 📈 Model Evaluation

The model was evaluated using:

- Accuracy Score
- Confusion Matrix
- Train-Test Split Validation
